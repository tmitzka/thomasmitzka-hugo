## General information on data protection 

The protection of your data is very important to me. That is why 
**thomasmitzka.at** only collects data that is necessary for the operation of the website 
or that you voluntarily provide.

As the **person responsible** for data processing on this website, I can be reached by
email:

Thomas Mitzka  
<tm@thomasmitzka.at>

## Automatically collected data

When you visit the website, certain data is automatically transmitted to the
web host **Netcup**, including, for example:

- IP address
- Date and time of access
- Browser and operating system
- Pages accessed

This data is **hosted in the EU**.

According to [information from Netcup](https://www.netcup.com/en/contact/data-privacy),
personal log files are stored for a **maximum of 14 days**, unless
longer storage is required.

## Web analysis

I use **[Simple Analytics](https://www.simpleanalytics.com)** to obtain anonymized
statistics about the use of the website.

Simple Analytics **does not set cookies** and **does not collect personal data**.
The data is **hosted in the EU**. Purpose of the analysis: to improve the website and
optimize its content.
